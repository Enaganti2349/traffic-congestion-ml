Traffic Congestion Prediction using Machine Learning

Overview
This project develops a machine learning model to predict traffic congestion levels using structured traffic and environmental data. The model classifies congestion into multiple levels based on input features.

Objectives
Perform data preprocessing and feature engineering
Train and evaluate a classification model
Analyze performance using standard evaluation metrics

Methodology
Data Cleaning and Handling Missing Values
Feature Engineering
Random Forest Classification
Model Evaluation using Accuracy and Confusion Matrix

Technologies Used
Python
Pandas, NumPy
Scikit-learn
Matplotlib
